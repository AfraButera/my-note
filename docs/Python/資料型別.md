### 1.整數 integer | int
```
year = int('2026')  <font color="#800080">將2026這個字串轉換成整數2026，並賦予給year變數</font>
print(year)         #輸出 year
print(type(year))   #輸出型別 <class 'int'>
```

### 2.浮點數（小數）float
````
a = 5              #a等於5
b = 3              #b等於3
c = 5 / 3          #c等於5除以3
print(c)           #輸出5除以3
print(type(c))     #輸出型別<class 'float'>
````

### 3.布林值 boolean | bool (Ture False)
