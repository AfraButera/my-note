# 英文學習紀錄

以 MJ English 為主教材，平日每天聽一集，聽完用 `engstart` 與 Claude 消化句型。

## 學習動機

英文不只是工具，而是一扇門。想用英文做到的事：

- **看脫口秀聽笑點** — 不靠字幕抓到幽默感和文化梗
- **讀外文書與詩集** — 有些文字的美是翻譯傳達不了的
- **不卡在技術文件上** — 官方 docs、論壇、GitHub issue 全都是英文，不想讓語言拖慢學習速度
- **日常對話能接得上** — 跟外國人說話不緊張
