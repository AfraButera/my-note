Docker 基本指令

拉取 Image#

從倉庫（如 DockerHub）拉取 image。tag 未指定時預設為 latest。

docker pull <imageName>:<tag>

建立新的 Container#

docker run 若本地無 image 會自動先 pull 再 run。

docker run <imageName># 或docker container run <imageName>

命名 Container：

docker run --name <containerName> <image>:<tag>

查詢指令說明#

docker

查看 Docker 版本#

docker -vdocker --version

查看 Container 狀態#

ps = process status，可查看所有 container（含未運行的）。

docker psdocker ps --alldocker ps -a

啟動 / 停止 / 重啟 Container#

docker start <containerID or containerName>docker stop <containerID or containerName>docker restart <containerID or containerName>

刪除 Container#

需先 stop，或加 -f 強制刪除正在運行的 container。

docker rm <containerID or containerName>docker rm -f <containerID or containerName>

查看本地 Images#

docker images

刪除 Image#

docker rmi <imageName>
