## range 需要先轉成 list 再放進 comprehension 嗎？

**Q：** 我寫了兩個 comprehension：先把 range 轉成 list，再從 list 篩選。這樣有問題嗎？

**A：** 能跑，但多了一步。`range` 本身就可以直接迭代，不需要先轉成 list：

```python
# 多餘的寫法
arr1 = [x for x in range(1, 51)]        # 先轉成 list
arr2 = [y for y in arr1 if y % 3 == 0]  # 再篩選

# 直接一步到位
arr2 = [x for x in range(1, 51) if x % 3 == 0]
```

list comprehension 的格式是 `[運算式 for 變數 in 可迭代物件 if 條件]`，`range` 就是可迭代物件，條件也可以直接寫在裡面，不需要分兩次。

## 切片 [1:-1] 是什麼意思？

**Q：** 習題二要去掉最大值和最小值，跟 list 切片有關嗎？

**A：** 有關，而且切片是這道題最乾淨的做法。

先把 list 排序，最小值就在最前面（index 0），最大值就在最後面（index -1）。用切片 `[1:-1]` 就是「從 index 1 開始，到倒數第二個結束」，剛好把頭尾都去掉：

```python
nums = [5, 3, 8, 1, 6]
nums.sort()          # [1, 3, 5, 6, 8]
trimmed = nums[1:-1] # [3, 5, 6]（去掉 1 和 8）
```

`-1` 在 Python 裡代表最後一個 index，所以 `[1:-1]` 就是「去頭去尾」的慣用寫法。
