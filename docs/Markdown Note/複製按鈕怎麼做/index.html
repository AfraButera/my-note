複製按鈕怎麼做

1.程式碼區塊：
#

大多數現代的 Markdown 編輯器或平台（如 GitHub、Stack Overflow、Notion、Obsidian）都會在程式碼區塊的右上角自動添加一個複製按鈕。你只需要使用「後撇號」（backticks）包裹代碼：

```python print("這段文字在很多平台上會自動出現複製按鈕，而且有加程式碼的標題框")```

2.Docusaurus支援MDX特性，Markdown製作React按鈕
#

在md檔寫直接寫：

<button   className="button button--primary"   onClick={() => navigator.clipboard.writeText('你要複製的內容')}>  複製特定文字</button>
