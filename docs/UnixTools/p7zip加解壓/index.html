    p7zip 加解壓縮

    7-Zip 是開放原始碼的壓縮程式，主要在 Windows 上運作。 p7zip 是 7-Zip 的 Unix/macOS 版本，透過 Homebrew 安裝。

    安裝 p7zip#

    brew install p7zip

    加密壓縮#

    7z a my_file -p -mhe=on
    參數說明
    aAdd（建立壓縮檔）
    -pPassword（加密，執行後會提示輸入密碼）
    -mhe=on隱藏壓縮檔內的檔名清單

    caution

    -mhe=on 只適用於 7z 格式，zip 格式不支援隱藏檔名，加上會報錯。

    zip 格式加密（不含隱藏檔名）：

    7z a my_file -tzip -p

    解壓縮#

    7z x my_file.7z

    執行後會提示輸入密碼。
